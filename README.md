# Workato Insights CSV Exporter

Workato Insights のテーブルウィジェットおよびチャート/KPI ウィジェットのデータを
CSV としてダウンロードする社内向け Chrome 拡張機能。

## 動作環境

- Google Chrome 102 以降
- Workato Insights ダッシュボードページ

## インストール

1. [最新版 zip をダウンロード](https://github.com/ryo168/WorkatoExtention/releases/download/v1.6.0/workato-insights-csv-v1.6.0.zip)
2. 任意のフォルダに解凍
3. Chrome で `chrome://extensions` を開く
4. 右上の「デベロッパーモード」を ON
5. 「パッケージ化されていない拡張機能を読み込む」をクリックし、解凍したフォルダを選択

## 使い方

1. Workato Insights のダッシュボードを開く
2. ツールバーの拡張機能アイコンをクリック
3. キャプチャ済みウィジェット一覧から対象を選び、CSV をダウンロード
